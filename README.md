# pizza-sales-analysis
This project analyzes pizza sales data using Python,SQL and Power BI to uncover business insights such as revenue trends, best-selling pizzas, and customer behavior.

---
# 🍕 Pizza Sales Data Analyst Portfolio Project

This project represents a complete, end-to-end data analytics workflow, simulating real-world business analysis tasks in a sales environment. It demonstrates how raw transaction data can be transformed into meaningful insights to support data-driven decision-making.

## 📌 This project is ideal for:
- Data Analyst interns building a strong portfolio  
- 📚 Learners practicing Python, SQL, and Power BI  
- 💼 Preparing for Data Analyst interviews  

---

## 📊 Project Overview

The objective of this project is to analyze pizza sales performance and extract actionable business insights through:

### 🔹 Data Preparation & EDA (Python)
- Cleaned and transformed raw sales data  
- Handled missing values  
- Prepared dataset for analysis  

### 🔹 Data Analysis (SQL)
- Calculated key KPIs: total revenue, average order value, total orders  
- Identified top-performing products, categories, and sizes  

### 🔹 Data Visualization (Power BI)
- Built interactive dashboard  
- Visualized trends and customer behavior  

### 🔹 Reporting & Insights
- Summarized findings  
- Provided business recommendations  

---

## 🔄 Project Workflow
Data Cleaning → Data Analysis → Data Visualization → Insights & Reporting

## 🛠️ How to Use This Project

1. Clone the repository  
2. Run Python Notebook  
   - Data cleaning  
   - Data exploration  
   - Prepare data for SQL  

3. Load Data into MySQL  
   - Create database  
   - Import dataset  
   - Run SQL queries  

4. Open Power BI Dashboard  
   - Load `.pbix` file  
   - Explore dashboard  

---

## 📈 Key Insights
- Classic category generated the highest revenue  
- Large size pizzas contributed most to total sales  
- Sales peak during weekends  
- A small number of products drive most revenue  

---

## 💡 Solution & Recommendations

- **Optimize product mix:** Focus on high-revenue categories like Classic pizzas  
- **Leverage peak periods:** Run promotions on weekends  
- **Promote large sizes:** Offer bundles or discounts  
- **Improve marketing:** Use top products in campaigns  
- **Customer insights:** Personalize offers to increase retention
